FICHIERS PYTHON (5 modules - 1,400+ lignes)
================================================================================

1. app.py (570 lignes)
   ├─ Interface Streamlit complète en français
   ├─ 2 modes: Manuel et Réel (avec données de marché)
   ├─ Sélection d'actions réelles
   ├─ Strikes suggérés automatiques
   ├─ 3-colonne layout avec paramètres, résultats, visualisations
   ├─ 2 graphiques interactifs (Payoff, Sensibilité volatilité)
   ├─ Tables d'analyse P&L
   └─ Affichage source des données (Yahoo Finance)

2. binomial_engine.py (209 lignes)
   ├─ Modèle Binomial Cox-Ross-Rubinstein
   ├─ Pricing d'options Call et Put
   ├─ Arbres binomiaux avec visualisation
   ├─ Calculs de facteurs u, d, q
   └─ Performance: ~5ms par call

3. strategy_manager.py (302 lignes)
   ├─ Classe ShortCondor avec logique 4-leg
   ├─ Payoff curves et breakevens
   ├─ Calculs de profit/perte maximum
   ├─ Classe StrategyExecutor pour gestion du capital
   ├─ Position sizing intelligent
   └─ Validation des strikes

4. market_data.py (150 lignes) ⭐ NOUVEAU
   ├─ Classe MarketDataProvider
   ├─ Récupération Yahoo Finance (yfinance)
   ├─ 10 actions majeures (AAPL, MSFT, GOOGL, etc.)
   ├─ Calcul volatilité historique (1 an)
   ├─ Prix automatique + suggestions de strikes
   ├─ Gestion d'erreurs robuste
   └─ Performance: <2 secondes par action

5. demo.py (260 lignes) + DEMO 6 RÉELLE
   ├─ 5 démos originales
   ├─ DEMO 6: Analyse mode réel avec AAPL
   ├─ Scénarios complets
   ├─ Gestion du capital
   ├─ Sensibilité volatilité
   └─ Nouveaux Greeks approximatifs

6. demo_real_mode.py (120 lignes) ⭐ NOUVEAU
   └─ Démo spécialisée pour mode réel uniquement

UTILISATION MAINTENANT
================================================================================

Lancer l'application:
  streamlit run app.py

Mode Réel - Étapes:
  1. Sélectionner "Mode Réel (Données de Marché)"
  2. Choisir une action (ex: AAPL)
  3. Données se chargent automatiquement
  4. Ajuster si nécessaire
  5. Analyser les résultats

Démo en ligne de commande:
  python demo.py
  (Scroll jusqu'à DEMO 6)

Démo Mode Réel uniquement:
  python demo_real_mode.py
